# BIGT token
This token is an ERC-20 token created using OpenZeppelin's ERC-20 contract. The BIGT token inherits all the functionalities of the ERC-20 standards
## functionalities: 
1. name
2. total supply
3. the decimal 
4. balance of 
5. transfer  
6. approve  
7. transferFrom
